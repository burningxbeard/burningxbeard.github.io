Donut Clicker

(HTML, CSS, Javascript)

Inspired by the Cookie Clicker™ game by Ortiel. The donut clicker project is built using HTML, CSS and Javascript. Clicking the main donut image makes 1 donut. Once you have made 10 donuts you can purchase your first auto-clicker which makes 1 donut every second. Each time an "auto-clicker" is purchased the price per auto-clicker rises by 10%. Follow the link and check it out.

MTG Deck Builder

(React, Typescript, CSS)

The MTG Deck Builder application is the result of a group project completed during the WeCanCodeIT Software Development bootcamp which I completed in October 2023. The application's front end was built using React and styled with CSS. For the backend my team used Node.js and Express along with a PostgreSQL database.
